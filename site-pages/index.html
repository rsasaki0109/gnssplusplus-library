S

Multi-Solver Engine

SPP, RTK (integer ambiguity via LAMBDA), PPP, and CLAS/MADOCA-PPP solvers with explicit state management and modular pipelines.

P

Protocol Agnostic I/O

RINEX, RTCM 2/3 (full SSR), UBX, QZSS L6, SBF, NMEA, NovAtel, and NTRIP. Auto-detection with protocol_detector.

V

Sign-off Framework

Machine-readable JSON summaries for every validation gate. Reproducible benchmark artifacts with CI-checked regression.

T

Comprehensive Testing

38 test files covering solvers, I/O protocols, ambiguity resolution, and slip detection. GTest for C++, Playwright for Web UI.

R

ROS 2 Integration

Native ROS 2 nodes for solution playback and raw GNSS driver. Publishes NavSatFix, PoseStamped, and Path messages.

D

Docker Ready

Multi-stage Docker builds for CI/CD. Compose-based web UI. Published to ghcr.io with branch and tag-based versioning.

Four Ways to Use

import libgnsspp

Web UI

Zero-dependency local dashboard for trajectories, benchmarks, and status.
